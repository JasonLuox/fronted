## 简介
vite 项目初始化时间为2022/2/19，所有依赖包基本上都是使用最新（latest）版本

搭建过程详解：[万字长文详解从零搭建企业级 vue3 + vite2+ ts4 框架全过程 - 掘金](https://juejin.cn/post/7069315908597973023)

## 主要依赖包版本一览

+ Vue ^3.2.25
+ Vite ^2.8.0
+ typescript ^4.5.4
+ eslint ^8.9.0
+ stylelint ^14.5.3
+ pinia ^2.0.11
+ echarts ^5.3.0
+ naive-ui ^2.25.5

## 安装

根目录下运行

```cmd
npm install
```

依赖包装好后

```cmd
npm run dev
```
